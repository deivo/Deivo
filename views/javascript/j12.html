【JavaScript】JS中==和===详解

Deivo 6/22/2015 JavaScript

# JS 中 ==

# 特点：

  • 两边比较之前，需将两边的数据类型转换程相同的类型，然后在比较。对于关系运算来说，会先将操作数据转化为原始值，在进行比较运算。

# 规律：

  1. NaN == NaN: NaN 和谁都不相等(包括自己)
  2. null == undefined //-> null 只有和 undefined 相等，和其他任何都不相等
  3. 对象 == 字符串 //-> 对象转化为字符串（对象.toString()）
  4. 剩下两边都不相等，都是转化成数字(对象转化为数字：Number(对象.toSting());
NaN == NaN //=> false
symbol(1) == symobol(1) //=> false
Infinity == Infinity //=> true
{} == {} //=> false 比较的是内存地址
  • 之前看到过一道有趣的面试题：考察的核心就是内置类的原型方法
var a = ?;
if(a == 1 && a == 2 && a==3) {
    console.log('ok');
}
// 上面a只有重写toString方法来实现相等
a = {
    i:0,
    toString: function() {
        return ++this.i;
    }
}

0 == null 结果为 false，为什么呢？ 原因：0 在内存中都会开辟一个位置，不管是栈内存，还是堆内存，而 null 在内存中都没有开辟位置，所以不相等。

# 特点：

  • 两边比较的数据类型相同且相等，即为 true，否则为 false。
